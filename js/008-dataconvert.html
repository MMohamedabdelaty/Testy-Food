Data Conversion and Coercion in JavaScript

Simply data type conversion is when we manually convert from one data type to another.

On the other hand data type coercion is when JavaScript automatically convert types from behind the scenes.

Best practice advice is to keep JavaScript coercion but when you got wrong results, So you need to figure out the data type causing the error and to convert it manually.

The Truthy and Falsy Values

The Falsy values are not exactly false but will become false when we convert them into a Boolean and these values in JavaScript are 5:

  1. 0
  2. '' or ""
  3. undefined
  4. null
  5. Nan

Any other values except the previous 5 are truthy and will become true when we convert them into a Boolean
