Data Types in JavaScript

We knew that variables contains values and these values in JavaScript can be classified into 2 main categories:

  1. Primitive Data Types
  2. Non-Primitive Data Type (Objects)
  1. Primitive Data Types
    • Numbers: Integral or Floating-point values.
    • String: Represents textual data. It can be enclosed in double, single, or backticks.
    • Boolean: This type holds two values: true or false, often used in logical operations. backticks.
    • Undefined: Indicates unassigned or not exist variables.
    • Null: To delete the assigned value without deleting the variable.
